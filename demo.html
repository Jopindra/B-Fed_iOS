System Overview
Navigation Structure:
• Dashboard Tab — 3 views: Today (Bottle), 7 Days (Wave), Month (Completion)
• History Tab — Chronological feed list
• FAB (+) — Log Feed modal

3-Period System:
• Today — Bottle visual, emotional reassurance
• 7 Days — Rhythm wave, short-term pattern
• Month — Completion calendar, consistency tracking

Modal Flow:
• Log Feed — Add new feed with timer
• Edit Feed — Modify or delete existing feed

Design Principles:
• No performance pressure
• Emotional, supportive tone
• Premium, calm aesthetics
• Clear visual hierarchy
